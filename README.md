## Name

demo-symfony6

## Description

Empty symfony6 php 8.x docker environment. With dabatase(mysql or postgres), nginx and portainer.io.

## Installation

Steps:

- git clone https://github.com/renerecalde/demo-symfony6.git
- cd demo-symfony6
- make build

Go to browser:

- http://localhost:250/index.php
